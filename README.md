# OCR Text Recognition Project

This project uses OpenCV and Tesseract OCR to extract text from images.

## Features
- Image loading
- Grayscale conversion
- Gaussian Blur
- Adaptive Thresholding
- Text extraction using Tesseract OCR
- Save extracted text to output.txt

## Technologies
- Python
- OpenCV
- PyTesseract
- NumPy
